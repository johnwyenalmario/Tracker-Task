# Tracker Task

# Submission to [Task Tracker] (https://roadmap.sh/projects/task-tracker)

## Installation
python task.py install

## Usage Guide
# 1. Add a New Task
task add "Buy groceries"

# 2. List Tasks
task list
task list todo
task list in-progress
task list done

# 3. Update a Task Description
task update 1 "Buy groceries and meal prep"

# 4. Track Progress
task mark-in-progress 1
task mark-done 1

# 5. Delete a Task
task delete 1
